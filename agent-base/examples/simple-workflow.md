# Simple Agent Workflow Example

This example shows how to use the simplified base structure for a small coding task.

## Example task

Add a dataset summary component to the app.

## Workflow

1. Use `planning` to define the goal and milestones
2. Use `tdd-workflow` to write a test first for the new component
3. Implement the component in `src/components/DatasetSummary.tsx`
4. Run the test and confirm it passes
5. Use `reviewer` to inspect the change and suggest improvements
6. Update documentation or examples if needed

## Notes

- keep the task scoped and incremental
- validate each step before moving to the next
- preserve the task plan and review notes in the repository
